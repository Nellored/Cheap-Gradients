# Simple Gradients
 A simple gradients in Garry's Mod

# Using 
 # Mat
 gU - Gradient Up
 gD - Gradient Down
 gR - Gradient Right
 gL - Gradient Left
 # X Position (Number)
 # Y Position (Number)
 # Width (Number)
 # Height (Number)
 # Color (rgb)
 GDN(mat, x, y, w, h, col)
# Example
    GDN(gU, 32, 32, 64, 64, Color(200, 100, 40))
